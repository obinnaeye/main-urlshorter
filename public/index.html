URL Shortener-Microservice
FreeCodeCamp API-Project By Nnenanya O.K

Root: https://urlshorterking.herokuapp.com
GitHub: https://github.com/obinnaeye/main-urlshorter

Sample 1

Input: https://urlshorterking.herokuapp.com/https://www.google.com.ng

Output: {
"originalURL": "https://urlshorterking.herokuapp.com/https://www.google.com.ng",
"shortURL": "https://urlshorterking.herokuapp.com/SyFdq7lBl"
}
And
https://urlshorterking.herokuapp.com/SyFdq7lBl
Will Redirect to:
https://www.google.com.ng
